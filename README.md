# 🐱 似てる動物AI - あなたにそっくりな動物を診断

![似てる動物AI](https://img.shields.io/badge/version-1.0.0-ff69b4) ![License](https://img.shields.io/badge/license-MIT-pink)

可愛いピンク系デザインの動物AI診断サイトです。顔写真をアップロードすると、AIがあなたに似ている動物を診断します！

## 🌸 プロジェクト概要

**似てる動物AI**は、ユーザーが写真をアップロードすることで、8種類の動物の中から最も似ている動物を診断する楽しいWebアプリケーションです。

### ✨ 主な特徴

- 🦔 **可愛いピンクデザイン** - 女性向けのポップで可愛らしいUI
- ⭐ **星のアニメーション** - 背景に輝く星がキラキラ動く
- 📷 **簡単診断** - 写真を撮るかアップロードするだけ
- 🎯 **8種類の動物** - ねこ、いぬ、きつね、うさぎ、たぬき、ハリネズミ、ハムスター、パンダ
- 📊 **似ている度表示** - パーセンテージバーでビジュアル表示
- 🌟 **動物の特徴説明** - 各動物の性格や特徴を詳しく解説
- 📱 **完全レスポンシブ** - スマホ・タブレット・PC対応
- 🔗 **SNSシェア機能** - Twitter/XとLINEで結果をシェア
- 🖼️ **画像ギャラリー** - 過去の診断結果を一覧表示（モザイク処理済み）

## 📂 プロジェクト構造

```
似てる動物AI/
```

- 🎭 モザイク処理（プライバシー保護）
- 🐱 動物絵文字と名前表示
- 8つのサンプル画像

## 🔧 使い方

### 基本的な使い方

1. **トップページを開く**
   - `index.html` をブラウザで開く

2. **写真をアップロード**
   - 「📷 カメラで撮影する」ボタンをクリック
   - カメラで撮影 or ギャラリーから選択

3. **診断結果を確認**
   - AIが2秒間診断
   - あなたに似ている動物と特徴が表示される

4. **結果をシェア**
   - Twitter/X または LINE でシェア

5. **画像一覧を見る**
   - トップページの「アップされた画像一覧を見る」リンクから

## 🎨 カスタマイズ方法

### 色の変更

`css/style.css` の以下の部分を編集：

```css
/* メインカラーの変更 */
.title {
    color: #FF1493; /* ここを変更 */
}

.upload-btn {
    background: linear-gradient(135deg, #FF1493 0%, #FF69B4 100%);
}
```

### 動物の追加

`js/main.js` の `animalData` 配列に追加：

```javascript
{
    name: '新しい動物',
    icon: '🦁',
    description: '説明文...',
    minSimilarity: 70,
    maxSimilarity: 95
}
```
